Proyecto desarrollado en spring boot 3.4.2

1.- Para levantar el proyecto debemos abrir la aplicación y correr el proyecto para ello debemos tener instalado docker compose

2.- Esta ejcución hará que se cree la imagen en docker compose

3.- Para revisar si se crearon las tablas correctamente, puedes abrir un programa de conexion a la base de datos en mi caso utilizo DBeaver y realizo la conexion como se encuentra
en el application. properties del proyecto.

4.- puedes probar con un cliente postaman la api tanto de registro como de login.
